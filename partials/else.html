The else keyword is used along with the if keyword to create if/else statements, which are how you direct Python to different things based on some condition. For example:

x = 1
if x < 10
  print "x is less than ten"
else:
  print "x is greater than ten"

When this code is run, the print "x is less than ten" statement will be run because 1 is less than 10. However, if the first line instead was x = 20, then the statements associated with the else block will be run and "x is greater than ten" will be printed instead.
