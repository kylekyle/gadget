The continue keyword is used to skip ahead to the next cycle of a while or for loop.


for x in range(1,10):
  if x == 5:
    continue
  print x

In the code above, the number 5 will not be printed. This is because the continue statement will execute when x is equal to 5. When that happens, the rest of the loop body, the print x line, will not be executed and the loop will restart with x equal to 6.
