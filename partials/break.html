The break keyword is used to exit a while or for loop before its original exit condition is met.


for x in range(3):
  print x
  if x == 1:
    break

In the code above, even though the for loop uses range(3), which should iterate 3 times, only 0 and 1 will be printed. This is because in the second iteration, x will be equal to 1 so the break statement will exit the loop early.
